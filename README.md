# MessGuard Hostel Management System

MessGuard Hostel Management System is a Java-based console application developed using Object-Oriented Programming (OOP). It helps manage hostel mess operations by allowing chefs to maintain menus and students to select meals while recording attendance and meal history through file handling.

---

## Features

-  Chef Login
-  Student Login
-  Add, Update, Delete & View Dishes
-  Breakfast, Lunch & Dinner Management
-  Automatic Student Attendance
-  Multiple Dish Selection
-  File Handling using `menu.txt` and `choices.txt`
-  Daily Attendance & Order Report

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling (Java I/O)
- ArrayList
- Scanner
- BufferedReader
- BufferedWriter
- PrintWriter

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/MessGuard-Hostel-Management-System.git
   ```

2. Open the project in NetBeans, IntelliJ IDEA, or Eclipse.

3. Compile and run the `Main.java` file.
