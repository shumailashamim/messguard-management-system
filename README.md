MessGuard Management System is a Java-based console application developed to simplify cafeteria and hostel mess operations. The system provides separate dashboards for chefs and students, allowing chefs to manage breakfast, lunch, and dinner menus while students can mark attendance and select meals. It implements CRUD operations, file handling for persistent data storage, and Object-Oriented Programming concepts including Abstraction, Inheritance, Encapsulation, and Polymorphism to create a structured and efficient management system.
